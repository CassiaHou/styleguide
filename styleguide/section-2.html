2.1Datepicker

The datepicker is based on bootstraps datepicker and all we do is wrap it and apply some special classes to make it more compatible with material design.

Example
From/to: 
<div class="wfm-datepicker-wrap">
<div class="sub-header">
	From/to:
</div>
<datepicker show-weeks="true" ng-model="someDate" class="wfm-datepicker"></datepicker>
</div >

2.2Calendar with range highlighted

Uses the 'custom class' with ui bootstrap datepicker to mark range.

Javascript example
$scope.setRangeClass = function (date, mode) {
if (mode === 'day') {
    var dayToCheck = new Date(date).setHours(12, 0, 0, 0);
    var startDay = new Date($scope.startDate).setHours(12, 0, 0, 0);
    var endDay = new Date($scope.endDate).setHours(12, 0, 0, 0);
    if (dayToCheck >= startDay && dayToCheck < endDay) {
        return 'range';
    }
}
return '';
};

Example
From:
To: 
<div class="wfm-datepicker-wrap">
<div class="sub-header">
	From:
</div>
<datepicker ng-model="startDate" max-date="endDate" class="wfm-datepicker"></datepicker>
</div>
<div class="wfm-datepicker-wrap">
<div class="sub-header">
	To:
</div>
<datepicker ng-model="endDate" min-date="startDate" custom-class="setRangeClass(date, mode)" class="wfm-datepicker"></datepicker>
</div>

2.3Line chart

A chart is a great way to present complex data. We use C3 charts to get nice features like legends, zoom and period selection.

Javascript example
var chart = c3.generate({
bindto: '#myChart',
data: {
    columns: [
        ['data1', 30, 200, 100, 400, 150, 250],
        ['data2', 20, 180, 240, 100, 190,0]
    ]
},
subchart: {
    show: true
},
zoom: {
    enabled: true
}
});

Example 
<div id="myChart"></div>
